# 📚 BookVerse

BookVerse — backend-платформа для организации книжных клубов, где пользователи могут объединяться, обсуждать книги и участвовать в голосованиях за следующее чтение.

✨ Основные возможности
👥 Регистрация и аутентификация пользователей (JWT)
🏛 Создание и управление книжными клубами
🔐 Ролевая модель доступа (владелец / участник)
📌 Вступление в клубы и выход из них
🚫 Модерация участников (бан / разбан)
🧑‍🤝‍🧑 Просмотр списка участников клуба со статусами
📚 Добавление книг в систему голосования
🗳 Проведение голосований за следующую книгу
📊 Получение результатов голосования
🧠 Защита API через Spring Security
🧱 Архитектура

Проект построен по классической многослойной архитектуре:

Controller — обработка HTTP-запросов
Service — бизнес-логика
Repository — доступ к данным (Spring Data JPA)
DTO + Mapper — изоляция доменной модели от API
🛠 Технологический стек
Java 17
Spring Boot
Spring Security
JWT
Spring Data JPA (Hibernate)
PostgreSQL
Maven
🔐 Безопасность
JWT-токены для аутентификации
Кастомный JwtFilter
Использование SecurityContextHolder
Контроль доступа на уровне эндпоинтов
🧩 Основные сущности
User — пользователь системы
Club — книжный клуб
ClubMember — связь пользователя и клуба (со статусом)
VoteSession — голосование
VoteOption — вариант выбора книги
